# KOPIS 공연예술통합전산망 데이터 수집기

공연예술통합전산망(KOPIS) Open API를 활용해 전체 데이터를 `.csv`로 수집하는 스크립트입니다.

## 수집 항목

### DB 검색
| 파일명 | 내용 |
|--------|------|
| `01_공연목록.csv` | 전체 공연 목록 (2010년~현재) |
| `02_공연상세.csv` | 공연별 상세 정보 |
| `03_공연시설목록.csv` | 공연 시설(극장) 목록 |
| `04_공연시설상세.csv` | 시설별 상세 정보 |
| `05_기획제작사목록.csv` | 기획·제작사 목록 |
| `06_수상작목록.csv` | 수상작 목록 |
| `07_축제목록.csv` | 축제 목록 |
| `08_원창작자목록.csv` | 원·창작자 목록 |

### 예매통계
| 파일명 | 내용 |
|--------|------|
| `09_예매상황판_일별.csv` | 일별 예매 현황 |
| `10_예매통계_기간별(주별).csv` | 주별 예매 통계 |
| `11_예매통계_월별.csv` | 월별 예매 통계 |
| `12_예매통계_장르별.csv` | 장르별 예매 통계 |

### 공연통계
| 파일명 | 내용 |
|--------|------|
| `13_공연통계_기간별.csv` | 기간별 공연 통계 |
| `14_공연통계_지역별.csv` | 지역별 공연 통계 |
| `15_공연통계_장르별.csv` | 장르별 공연 통계 |
| `16_공연통계_공연별.csv` | 공연별 통계 |
| `17_공연통계_시설별.csv` | 공연 시설별 통계 |
| `18_공연통계_가격대별.csv` | 가격대별 통계 |

## 사용 방법

### 1. 패키지 설치
```bash
pip install requests
```

### 2. API 키 설정
`kopis_collector.py` 상단의 `API_KEY` 변수에 발급받은 키를 입력하세요.

```python
API_KEY = "여기에_API_키_입력"
```

### 3. 실행
```bash
python kopis_collector.py
```

수집된 CSV 파일은 `C:\Users\82106\Desktop\Claude\kopis_data\` 폴더에 저장됩니다.

## 주의사항
- 공연상세(02번)와 공연시설상세(04번)는 건수가 많아 수집에 시간이 걸릴 수 있습니다.
- API 과부하 방지를 위해 요청 간 0.3초 딜레이가 설정되어 있습니다.
- KOPIS API는 XML 형식으로 응답하며, 스크립트가 자동으로 CSV로 변환합니다.

## API 출처
- [공연예술통합전산망 Open API](https://www.kopis.or.kr/por/cs/openapi/openApiInfo.do)
